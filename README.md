# Product Management System

A full-stack CRUD application built using React and Spring Boot.

## Tech Stack
- Frontend: React, Bootstrap, Axios
- Backend: Spring Boot, REST APIs, JPA
- Database: PostgreSQL
- Tools: Git, GitHub

## Project Structure
- frontend/ → React application
- backend/ → Spring Boot application

## Features
- Add product
- View products
- Update product
- Delete product

## How to Run Locally

### Backend
1. Open backend folder in Eclipse/IntelliJ
2. Configure database in `application.properties`
3. Run Spring Boot application

### Frontend
```bash
cd frontend
npm install
npm start
